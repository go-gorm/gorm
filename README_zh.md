## 总览

* 全功能ORM
* 关联（具有一个，具有多个，属于，一对多，多态性，单表继承）
* 钩子（创建/保存/更新/删除/查找之前/之后）
* 希望通过 `Preload` , `Joins` 加载
* 事务，嵌套事务，保存点，回滚到保存点
* 上下文、准备状态模式、运行模式
* 批量插入、批量查询、地图查找
* SQL生成、更新插入、锁定、优化器/索引/注释提示
* 复合主键
* 自动迁移
* 日志
* 可扩展，基于GORM回调编写插件
* 每个功能都附带测试
* 对开发人员友好
